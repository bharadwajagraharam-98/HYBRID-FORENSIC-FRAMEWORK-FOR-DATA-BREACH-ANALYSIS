# Hybrid Forensic Framework for Data Breach Analysis

## Overview
This project presents a hybrid forensic framework integrating:

- Disk Forensics (Autopsy)
- Memory Forensics (Volatility 3)
- Machine Learning
- DikeDataset Malware Analysis

The framework improves detection of:
- Fileless malware
- Insider threats
- Suspicious package behavior
- Memory-resident attacks

## Technologies Used
- Python
- Scikit-learn
- TensorFlow
- PyCaret
- Autopsy
- Volatility 3
- Jupyter Notebook

## Dataset
DikeDataset:
https://github.com/iosifache/DikeDataset

## Features
- Malware clustering using DBSCAN
- Malice score distribution
- Event log reconstruction
- Disk and memory forensic analysis
- Behavioral anomaly detection
